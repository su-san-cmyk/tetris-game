# テトリス風ゲーム

Vite + Vanilla JavaScript + HTML5 Canvas で作られたテトリス風ブロックパズルゲームです。

## 起動方法

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev

# 本番ビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## キャラクター紹介

### 💎 エメラ（バランス型）
スタンダードな性能。初めての人にもベテランにもおすすめ。特殊能力なし。

### 🌊 アクア（初心者向け）
ゆっくり落下するのでじっくり考えられる。落下速度0.7倍・スコア倍率0.8倍。

### 🔥 フレイム（攻撃型）
ハードドロップとテトリスで高スコアを狙える上級者向け。ハードドロップ1.5倍・テトリス1.3倍・落下速度1.1倍。

### 🌑 シャドウ（テクニカル型）
壁蹴り判定が広く、接地後の猶予も長い。壁蹴り判定2倍・接地猶予1.5倍。

## 機能

- SRS（スーパーローテーションシステム）準拠の回転・壁蹴り
- 7-bag ランダム方式によるミノ生成
- ゴーストピース表示
- NEXT 3個先読み表示
- HOLD 機能
- レベルシステム（10ライン毎にレベルアップ）
- ハイスコア保存（localStorage）
- 設定画面（BGM/SE音量・ミュート）
- ポーズ機能

## 音声ファイルの追加

`public/sounds/` ディレクトリに以下のファイルを配置することで、BGMと効果音が再生されます。

| ファイル名 | 用途 |
|-----------|------|
| `bgm.mp3` | ゲーム中のBGM（ループ再生） |
| `drop.mp3` | ピース設置音 |
| `rotate.mp3` | 回転音 |
| `line-clear.mp3` | ライン消去音 |
| `gameover.mp3` | ゲームオーバー音 |

音声ファイルが存在しない場合は無音でプレイできます。

## 技術スタック

- **Vite** - ビルドツール・開発サーバー
- **Vanilla JavaScript (ES Modules)** - フレームワークなし
- **HTML5 Canvas** - ゲーム描画
- **CSS カスタムプロパティ** - テーマカラー管理
